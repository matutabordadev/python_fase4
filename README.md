# Sistema de Promedios de Notas

Programa en Python que permite calcular el promedio de notas de un alumno a través de un menú interactivo por consola, validando que las notas ingresadas sean numéricas y estén dentro de un rango válido.

## Funcionalidades

- Menú interactivo por consola
- Carga de notas con validación (solo números entre 1 y 10)
- Cálculo automático del promedio
- Determina si el alumno aprueba o desaprueba
- Permite realizar múltiples cálculos sin reiniciar el programa

## Cómo usar el programa

1. Ejecutar el archivo `main.py`
2. Ingresar el nombre del usuario
3. Seleccionar la opción **Calcular promedio**
4. Ingresar las notas solicitadas
5. Visualizar el promedio final y el estado del alumno

## Reglas de aprobación

- El promedio se calcula sobre 4 notas
- El alumno aprueba con promedio mayor o igual a 6
- Las notas válidas deben estar entre 1 y 10

## Objetivo del proyecto

Proyecto realizado con fines educativos para practicar:
- Uso de funciones (`def`)
- Retorno de valores (`return`)
- Validación de datos con `try / except`
- Manejo de bucles y condicionales
- Organización de un programa en Python

## Tecnologías utilizadas

- Python 3
